Network Optimization Project

Optimizing a network for better performance and lower latency.

By Sakshi Rane

Project Overview

This project involved optimizing the network infrastructure of a small business to reduce latency and improve overall performance. The goal was to streamline communication between the server and client devices, reducing delays and improving reliability.

Technologies Used

  • Python
  • Network simulation tools (GNS3, Cisco Packet Tracer)
  • Routing and switching configurations
  • Network performance monitoring

Key Outcomes

  • Reduced latency by 30% by optimizing the network topology and using efficient routing protocols.
  • Improved bandwidth allocation to ensure critical applications receive the required resources.
  • Enhanced network security by implementing firewall rules and VPN connections.

Challenges and Solutions

One of the major challenges was dealing with outdated hardware. The solution involved reconfiguring network devices to ensure that the infrastructure could support newer software without needing expensive upgrades.
